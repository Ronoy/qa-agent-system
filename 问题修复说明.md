# 问题修复说明

## 问题 1：图片无法解析 ✅ 已修复

**问题描述**：
上传图片文件后，后端只返回 `[IMAGE:文件名]`，没有进行 OCR 文字识别，导致图片内容为空无法分块和检索。

**根本原因**：
`backend/app/services/kb_service.py:54` 对图片类型文件没有实现 OCR 功能。

**解决方案**：
1. 添加 `extract_image()` 函数，使用 Tesseract OCR 识别图片文字
2. 支持中英文混合识别（chi_sim+eng）
3. 安装依赖：
   - Python 库：pytesseract（已安装）
   - 系统工具：tesseract + tesseract-lang（安装中）

**修改文件**：
- `backend/app/services/kb_service.py` - 添加图片 OCR 功能
- `backend/requirements.txt` - 添加 pytesseract 依赖

**测试方法**：
```bash
# 等待 Tesseract 安装完成后
brew list tesseract

# 重启后端
./start_backend.sh

# 上传图片测试
curl -X POST http://localhost:8000/api/knowledge/bases/{kb_id}/upload \
  -F "file=@test.jpg"
```
